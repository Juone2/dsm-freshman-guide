## dsm-freshman-guide

대덕SW마이스터고등학교 신입생을 위한 전공 공부 가이드입니다.<br>
**DSM 학생이라면 누구나 pull request**를 통해 내용을 추가하고 변경할 수 있습니다.

## 프로젝트 실행 방법

### 툴 설치
- window
  - https://nodejs.org/en/download에서 node.js 설치
  - pnpm 설치
    ```bash
    npm install -g pnpm
    ```
- macOS
  - node.js 및 pnpm 설치
    ```bash
    brew install node
    npm install -g pnpm
    ```

### 실행
```bash
pnpm install
pnpm run dev
```
